# Michael-Popaj-Sudoku-Puzzle-Generator

Sudoku Puzzle Generator

By Michael Popaj

This Java Console program generates a fully solved 9x9 Sudoku board. 

In order to run this program, you should download the .java file and use a software like Java Sandbox to access the SudokuProjectGenerator class and run it as a console program with a fully formatted output. 

The program begins by initializing a blank 9x9 grid. The Sudoku board is generated using a recursive backtracking algorithm to place the proper numbers. Alongside this, it uses a Fisher-Yates shuffle to ensure number uniqueness. It then prints the result with clear and concise formatting, including 3x3 box separators.
